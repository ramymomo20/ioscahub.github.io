FastDL Asset Host

What This Page Is For

FastDL hosts the exact custom files your Source server references so players download them over HTTP instead of trickle-downloading from the game server.

Mirror server paths exactly Keep file name casing exact Publish .bz2 beside raw files

IOSoccer config example

sv_allowdownload 1
sv_allowupload 1
sv_custom_assets 1
sv_downloadurl "https://ioscahub.github.io/fastdl/iosoccer/"

Point each server at its own game folder URL, not at the landing page itself.

Folder Rules

  • For IOSoccer, enable sv_custom_assets 1 in iosoccer\cfg\autoexec.cfg.
  • The web path must match the server path exactly.
  • Wrong casing will cause HTTP download fallback.
  • Use the trailing slash in sv_downloadurl.
  • Test a real asset URL in a browser after publish.

Common Folders

Folder Typical files Example URL
maps .bsp, .nav, overview text /fastdl/iosoccer/maps/custom_arena.bsp.bz2
materials .vmt, .vtf /fastdl/iosoccer/materials/custom/banner.vtf.bz2
models .mdl, .vvd, .phy, .vtx /fastdl/iosoccer/models/player/custom.mdl.bz2
sound .wav, .mp3 /fastdl/iosoccer/sound/custom/join.mp3.bz2
resource .txt, language or UI assets /fastdl/iosoccer/resource/ui/custom.txt.bz2

Local Build Command

python scripts/build_fastdl.py `
  --source "D:\Servers\IOSoccer\iosoccer" `
  --site-root "iosca_hub_github\ioscahub.github.io\fastdl" `
  --game-dir "iosoccer" `
  --clean

The script copies the usual FastDL folders into this site and writes a .bz2 copy beside each file so the published site is immediately usable for Source clients.
